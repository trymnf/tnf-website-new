Solving Advent of Code using tidyverse

02 Dec, 2025

When I find the time, I like to try solving the Advent of Code puzzles. I prefer to take a tidy, data-like approach wherever possible (because that’s all I know). Let’s see how far that gets me this year.

Day 2 asks us to find numbers that contain patterns (like 99 or 1188511885) within ranges of numbers. This task is very friendly to a tidyverse approach.

The separate functions from tidyr and a little regex made today’s solution clean and straightforward.

 1library(tidyverse)
 2
 3input <- "11-22,95-115,998-1012,1188511880-1188511890,222220-222224,1698522-1698528,446443-446449,38593856-38593862,565653-565659,824824821-824824827,2121212118-2121212124"
 4
 5data <- tibble(input = input) |>
 6  separate_longer_delim(input, ",") |>
 7  separate(input, into = c("start", "end"), sep = "-") |>
 8  rowwise() |>
 9  mutate(range = list(start:end)) |>
10  unnest_longer(range)

range here gives us all the values in all the ranges. Now it’s just a matter of finding the repetitions using regex, and adding them all together.

1 data |>  
2  filter(str_detect(range, "^([0-9]+)\\1$")) |>
3  summarise(sum(range))

# A tibble: 1 × 1
  `sum(range)`
         <int>
1   1227775554

For part 2, all I needed to add was a + to the regex to allow for more than one repetition of the pattern.

 1# Part 2
 2
 3tibble(input = input) |>
 4  separate_longer_delim(input, ",") |>
 5  separate(input, into = c("start", "end"), sep = "-") |>
 6  rowwise() |>
 7  mutate(range = list(start:end)) |>
 8  unnest_longer(range) |>
 9  # adding a plus here for unlimited repeats
10  filter(str_detect(range, "^([0-9]+)\\1+$")) |>
11  summarise(sum(range))

# A tibble: 1 × 1
  `sum(range)`
         <dbl>
1   4174379265
